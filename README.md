# Bedrock — เรียน Python จากศูนย์

เว็บแอปสื่อการเรียนรู้การเขียนโปรแกรม Python สำหรับผู้เริ่มต้นที่ไม่เคยเขียนโค้ดมาก่อน
หัวใจของแอปคือทำให้ผู้เรียนรู้สึก **"ฉันทำได้"** ตั้งแต่บรรทัดแรก ไม่กลัว ไม่ท้อ และจบทุกด่านด้วยชัยชนะ

## เป็นอะไร

- บทเรียน Python 15 ด่าน เรียงจากง่ายไปยาก ทีละ 1 แนวคิดต่อด่าน
- มินิโปรเจกต์ 4 ชิ้น + โปรเจกต์ใหญ่ (Capstone) ที่ผู้เรียนเลือกหัวข้อเอง
- เขียนและรันโค้ด Python ได้จริงในเบราว์เซอร์ (ไม่ต้องติดตั้งอะไร)
- ข้อความผิดพลาด (error) แปลเป็นภาษาไทยเข้าใจง่าย โทนเป็นมิตร 18 รูปแบบ
- ระบบคำใบ้แบบเป็นขั้น + แผนที่เส้นทางหินที่เห็นความก้าวหน้าชัดเจน
- เก็บความก้าวหน้าไว้ในเครื่อง (localStorage) ไม่มีการสมัครสมาชิก ไม่มีการติดตามตัว

## วิธีเปิดใช้

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ได้เลย — ไม่ต้องติดตั้งหรือ build อะไรทั้งสิ้น

> หมายเหตุ: ฟอนต์ ตัวแก้ไขโค้ด (CodeMirror) และตัวรัน Python (Skulpt) โหลดผ่าน CDN
> ดังนั้นเครื่องควรต่ออินเทอร์เน็ตในการใช้งานครั้งแรก

## เทคโนโลยีที่ใช้

- **HTML5 + CSS3 + JavaScript (Vanilla)** — ไม่มี framework ไม่มี build tool ดูแลแก้ไขเองได้
- **[Skulpt](https://skulpt.org/)** — รันโค้ด Python ในเบราว์เซอร์ (ผ่าน CDN)
- **[CodeMirror 5](https://codemirror.net/5/)** — ตัวแก้ไขโค้ดพร้อมเลขบรรทัดและไฮไลต์ (ผ่าน CDN)
- **localStorage** — เก็บความก้าวหน้าในเครื่องผู้ใช้ (ไม่มี backend / ฐานข้อมูล)

## โครงสร้างไฟล์

```
Learning Platform/
├── index.html            หน้าต้อนรับ
├── map.html              แผนที่เส้นทางหิน (เลือกด่าน)
├── lesson.html           หน้าเรียน 1 ด่าน (โหลดตาม ?id= หรือ ?project=)
├── css/
│   └── style.css         ระบบดีไซน์ + design tokens
├── js/
│   ├── config.js         BEDROCK_CONFIG — ศูนย์รวมค่าพฤติกรรมทั้งหมด
│   ├── storage.js        จัดการความก้าวหน้าใน localStorage
│   ├── errors.js         แปล error เป็นภาษาไทย 18 แบบ
│   ├── missions.js       เนื้อหา 15 ด่าน + มินิโปรเจกต์ + Capstone
│   ├── runner.js         รันโค้ดด้วย Skulpt + จับ error
│   ├── hints.js          ระบบคำใบ้แบบเป็นขั้น
│   └── app.js            ตรรกะรวม + การนำทาง + แผนที่ + โมดอลชนะ
├── assets/
└── README.md
```

## ปรับแต่งพฤติกรรม

ค่าพฤติกรรมทั้งหมด (ความยาก จำนวนคำใบ้ เวลาจำกัดการรัน สี error ฯลฯ) อยู่รวมกันใน
`js/config.js` ที่ตัวแปร `BEDROCK_CONFIG` — แก้ที่เดียว มีผลทั้งแอป

## วิธี deploy ขึ้น Vercel

1. push โค้ดทั้งโฟลเดอร์นี้ขึ้น GitHub repository
2. เข้า [vercel.com](https://vercel.com) แล้วกด **Add New > Project**
3. เลือก **Import** repository ที่เพิ่ง push
4. ไม่ต้องตั้งค่า build (เป็นเว็บ static ล้วน) กด **Deploy** ได้เลย
5. เปิดลิงก์ที่ Vercel ให้มา ก็ใช้งานได้ทันที

เนื่องจากเป็นเว็บ static ล้วน จะนำไปวางที่ GitHub Pages, Netlify หรือโฮสต์ static ใด ๆ ก็ได้เช่นกัน
